# Food Ordering System

## Overview
This is a food ordering system developed as a first-year university project.  
The system allows customers to browse food items, place orders, and track order status, while admins can manage menu items and orders.

## Features
- User registration and login
- Browse food menu
- Place orders
- Admin panel for managing food items and orders

## Technologies Used
- Programming Language: Java 
- Database: MySQL / SQL Server 


## How to Run the Project
1. Clone the repository
2. Import the project into your IDE
3. Set up the database using the provided SQL file
4. Run the application
